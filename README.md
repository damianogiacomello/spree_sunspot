Spree::Sunspot
==============

Use Sunspot as search engine in Spree applications


Basic Installation
------------------

1. Add the following to your Gemfile
<pre>
  gem 'spree_sunspot', '~> 1.0.0'
</pre>
2. Run `bundle install`
3. lunch installation:
<pre>
	rails g spree_sunspot:install
</pre>
